# Конфигурация проекта

## Структура

```
config/
├── envs/                 # .env файлы с переменными окружения
├── yaml/                 # .yaml файлы со структурными настройками
├── modules/              # Python модули с Pydantic моделями (БД, Redis, JWT)
└── settings/             # Основные настройки приложения
```
## envs/ — Переменные окружения

|      Файл      | Назначение |
|----------------|------------|
|     `.env`     | Локальные настройки (не коммитится) |
| `.env.example` | Пример переменных (коммитится) |


## yaml/ — YAML конфигурации

|      Файл      | Назначение |
|----------------|------------|
|  `.base.yaml`  | Базовые настройки |
|  `local.yaml`  | Локальные переопределения (не коммитится) |


## modules/ — Pydantic модели

|     Файл    |    Назначение      |
|-------------|--------------------|
| database.py |    Настройки БД    |
|    app.py	  | Основные настройки |


## Приоритет загрузки
Переменные окружения
.env файл
YAML файл
Значения по умолчанию
